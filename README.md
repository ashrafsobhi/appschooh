# نظام شؤون الطلبة

نظام إداري متكامل لإدارة بيانات الطلبة والتسجيلات باستخدام Django.

## المميزات

- **صفحة الترحيب**: عرض إحصائيات سريعة عن الطلبة
- **صفحة قيد الطلبة**: عرض جدول شامل لجميع الطلبة مع إمكانية البحث والتصفية
- **صفحة إضافة طالب**: إضافة طالب يدوياً أو من خلال رفع ملف Excel

## البيانات المدعومة

- كود الطالب
- اسم الطالب
- الديانة
- التبعية
- حالة القيد
- نوع القيد
- الشعبة
- النوع (ذكر/أنثى)
- محافظة الميلاد
- تاريخ الميلاد
- الرقم القومي
- السن
- السن عند الامتحان
- المدرسة
- اسم ولي الأمر

## متطلبات التشغيل

- Python 3.8 أو أحدث
- Django 4.2 أو أحدث
- openpyxl (لقراءة ملفات Excel)

## التثبيت والتشغيل

1. **تثبيت المتطلبات**:
```bash
pip install -r requirements.txt
```

2. **إنشاء قاعدة البيانات**:
```bash
python manage.py makemigrations
python manage.py migrate
```

3. **إنشاء مستخدم إداري** (اختياري):
```bash
python manage.py createsuperuser
```

4. **تشغيل السيرفر**:
```bash
python manage.py runserver
```

5. **الوصول للنظام**:
- افتح المتصفح على: `http://127.0.0.1:8000`
- لوحة التحكم: `http://127.0.0.1:8000/admin`

## استخدام ملف Excel

عند رفع ملف Excel، يجب أن يحتوي على الأعمدة التالية بالترتيب:

1. كود الطالب
2. اسم الطالب
3. الديانة (إسلام، مسيحية، يهودية، أخرى)
4. التبعية (مصري، أجنبي)
5. حالة القيد (نشط، غير نشط، معلق، متخرج)
6. نوع القيد (عادي، منقول، مكرر)
7. الشعبة
8. النوع (ذكر، أنثى)
9. محافظة الميلاد
10. تاريخ الميلاد (صيغة: YYYY-MM-DD)
11. الرقم القومي
12. السن
13. السن عند الامتحان
14. المدرسة
15. اسم ولي الأمر
