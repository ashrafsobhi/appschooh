📋 خطوات التشغيل:

1️⃣  تثبيت المتطلبات:
   pip install -r requirements.txt

2️⃣  إنشاء قاعدة البيانات:
   python manage.py makemigrations
   python manage.py migrate

3️⃣  إنشاء مستخدم إداري (اختياري):
   python manage.py createsuperuser

4️⃣  تشغيل السيرفر:
   python manage.py runserver

5️⃣  فتح المتصفح:
   http://127.0.0.1:8000
